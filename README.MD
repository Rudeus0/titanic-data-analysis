# Titanic # Titanic Data Analysis & Insight Report

### 🎯 Objective
To perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify key survival factors using probabilistic reasoning and statistical visualization.

### 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas (Data Manipulation), Matplotlib (OO-style Visualization)
* **Environment:** VS Code / Jupyter

### 🧪 Key Engineering Steps
1. **Data Cleaning:** Handled missing values using median (Age) and mode (Embarked) to maintain distribution integrity.
2. **Feature Engineering:** Created `FamilySize` to analyze the impact of social units on survival probability.
3. **Visualization:** Used the `fig, ax` (Object-Oriented) approach for precise control over plot aesthetics.

### 📊 Key Insights
* **Gender Bias:** Survival rate for females was significantly higher (~74% vs ~18%).
* **Class Hierarchy:** Pclass 1 passengers had a higher probability of survival, likely due to cabin location and evacuation priority.
